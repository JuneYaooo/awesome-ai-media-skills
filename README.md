## FAQ

### What is a Claude Code Skill?

A Claude Code Skill is a reusable capability that extends Claude Code's functionality. Skills can be invoked via slash commands and provide specialized domain knowledge and tooling for specific tasks like video generation or social media management.

### What is an MCP Server?

MCP (Model Context Protocol) servers provide tools that AI assistants can call to interact with external systems. For media workflows, MCP servers can connect to social media APIs, video editing tools, and content management systems.
